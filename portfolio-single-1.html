GalaxyGaze

Project Date May 5th, 2024
Role Hosted Nasa APOD Website
Visit GitHub

GalaxyGaze is a Node.js project developed using React and Bootstrap for the front-end, featuring Firebase for authentication and database management. Users can securely log in with their Gmail accounts, allowing for a personalized experience. The application allows users to explore stunning Mars Rover images based on specific dates they select, providing detailed information about each image to enhance their understanding of Mars exploration.

The primary goal of GalaxyGaze is to offer an intuitive interface for users to engage with Mars Rover data. By collecting user details like name and email, the project aims to foster a deeper appreciation for space exploration and facilitate seamless access to captivating visuals from Mars.
